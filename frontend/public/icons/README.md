# PWA Icons

## Generate Icons

Use one of these tools to generate all required icon sizes:

### Option 1: Online Generator
1. Visit: https://realfavicongenerator.net/
2. Upload your logo (512x512 PNG recommended)
3. Download generated icons
4. Extract to this folder

### Option 2: CLI Tool
```bash
npx pwa-asset-generator logo.png ./public/icons --icon-only
```

## Required Sizes
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## For Demo
Icons are optional. The app works without them (browser will use default).
